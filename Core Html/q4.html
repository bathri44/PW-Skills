Ordered lists of HTML tags

  1. Head tag

    Head tag gives meta information about the HTML page.

  2. Title tag

    This tag gives the title of the HTML page.

  3. Body tag

    This tag gives all the visible contents like, headings, paragraph, images, video, audio etc.

  4. Heading tag (h1 to h6)

    It gives the headings in variable size and importance. h1 is most important and h6 is the least.

  5. Paragraph tag (p)

    It denotes or gives that the content inside this tag is a paragraph.

  6. Anchor tag (a)

    This tag defines a hyperlink, which is used to link from one page to another. the href attribute, which indicates the link's destination.
